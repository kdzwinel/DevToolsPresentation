What's DevTools?

  • based on the WebKit Web Inspector
  • it's a web-app
  • integrated, not an add-on
  • getting quite popular (dailyJS survey 2012)

How to debug my CSS and HTML?

Working with styles

  • opening, positioning, inspecting, moving around the DOM, searching, undoing
  • "Styles" pane
    • tabbing & auto-completion
    • modifying the value with arrows
    • add new style
    • color switch & color picker
    • toggling element state
  • "Computed Styles" pane
  • "Metrics" pane
  • revision history, saving to disk

Remote debugging 

  • adb forward tcp:9222 localabstract:chrome_devtools_remote
  • chrome.exe --remote-debugging-port=9222
  • WebKit Remote Debugging Protocol
  • PonyDebugger, Node Inspector
  • inspecting iPhone from Chrome ? Safari's Web inspector does use the WebKit remote debugging protocol. However, Safari does not use TCP/HTTP as a transport layer, thus making it incompatible with Chrome.
